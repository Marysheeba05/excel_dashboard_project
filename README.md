# Excel Dashboard Project

Interactive Excel dashboard created using pivot tables, charts, slicers, and formulas to visualize and analyze business data.

# Project Overview
This project is an Excel dashboard built within a single workbook.

# Workbook Structure
- Sheet 1: Raw data with Excel formulas
- Sheet 2: Pivot tables used for analysis
- Sheet 3: Interactive dashboard with charts and slicers

# Tools & Features Used
- Excel formulas
- Pivot Tables
- Charts and Slicers
